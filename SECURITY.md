# Security Policy

## Supported Versions

We actively provide security updates for the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please send an e-mail to **security@ghostcompiler.com**.

All security vulnerabilities will be promptly addressed. Please do not open public GitHub issues for security vulnerabilities.
